# MERN Movie Booking Website

A full stack movie ticket booking web application built using the MERN stack.  
Users can browse movies, select show dates and times, book seats, and pay using UPI.

---

## Features

- User authentication (Login / Signup)
- Browse available movies
- Select show date and show time
- Seat selection system
- Real-time seat availability
- UPI payment with QR code
- Booking history page

---

## Tech Stack

Frontend:

- React
- Vite
- Tailwind CSS

Backend:

- Node.js
- Express.js

Database:

- MongoDB
